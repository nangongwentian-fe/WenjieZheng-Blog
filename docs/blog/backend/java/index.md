# Java

Java 是一种广泛使用的面向对象编程语言，特别适合企业级应用开发。

## 特性

- 跨平台（Write Once, Run Anywhere）
- 面向对象
- 强类型语言
- 自动内存管理
- 丰富的生态系统
- 高性能

## 核心技术栈

### 基础框架
- Spring Boot - 快速构建应用
- Spring Cloud - 微服务开发
- Spring MVC - Web 开发
- MyBatis / JPA - 持久层框架

### 中间件
- Redis - 缓存
- RabbitMQ / Kafka - 消息队列
- Dubbo - RPC 框架

### 开发工具
- Maven / Gradle - 构建工具
- IntelliJ IDEA - 开发IDE
- JUnit - 单元测试

## 应用场景

- 企业级应用开发
- 微服务架构
- 大数据处理
- Android 应用开发
- Web 应用开发

## 开始学习

推荐从 Spring Boot 开始学习现代 Java 后端开发。
