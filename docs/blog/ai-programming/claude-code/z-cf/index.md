# ZCF - Zero-Config Code Flow：零配置 Claude Code 环境配置工具

## 项目概述

ZCF（Zero-Config Code Flow）是一个专为 Claude Code 设计的零配置环境设置工具，旨在为开发者提供一键式的 Claude Code 和 Codex 环境配置解决方案。该项目支持中英文双语配置、智能代理系统和个性化 AI 助手，极大地简化了 AI 辅助编程的环境搭建过程。

> **项目地址**：[UfoMiao/zcf](https://github.com/UfoMiao/zcf)

## 核心特性

### 🚀 零配置安装体验

ZCF 的核心理念是"零配置，一键搞定"。通过交互式菜单系统，用户可以轻松完成：

- **自动检测和安装**：智能检测 Claude Code 安装状态并自动安装
- **多语言支持**：支持中文、英文、日文界面和配置文件
- **智能备份机制**：自动备份现有配置，确保数据安全
- **跨平台兼容**：完美支持 Windows、macOS、Linux、WSL 和 Termux

### 🎯 交互式菜单系统（v2.0 核心功能）

```bash
npx zcf  # 打开交互式菜单
```

菜单提供的主要功能包括：

1. **完整初始化**：一键完成 Claude Code 安装 + 工作流导入 + API 配置 + MCP 服务设置
2. **工作流更新**：快速更新 AI 工作流和命令系统
3. **API 配置**：支持官方登录、Auth Token、API Key 和 CCR 代理
4. **MCP 服务配置**：可视化配置各种 MCP 服务
5. **默认模型设置**：灵活配置 Opus、Sonnet、Sonnet 1M 或自定义模型
6. **AI 记忆配置**：个性化 AI 输出语言和全局输出风格

### 🤖 Codex 支持（v3.0.0 重磅功能）

ZCF v3.0.0 引入了对 OpenAI 官方 Codex CLI 的完整支持，实现：

- **统一工具管理**：在 Claude Code 和 Codex 之间无缝切换
- **多提供商支持**：配置多个 API 端点，支持提供商切换
- **智能配置系统**：自动 Codex CLI 安装和 MCP 服务集成
- **系统提示集成**：安装专业 AI 个性（工程师、猫娘工程师、老王工程师）

### 🎨 AI 输出风格（v2.12+ 个性化功能）

ZCF 支持多种预设的 AI 输出风格，让开发者能够个性化 Claude Code 的交互体验：

## 安装和使用

### 快速开始

```bash
# 推荐方式：使用交互式菜单
npx zcf

# 直接完整初始化
npx zcf i  # 或 npx zcf init

# 仅更新工作流
npx zcf u  # 或 npx zcf update
```

### 多语言支持

ZCF 支持完整的多语言体验：

```bash
# 中文界面
npx zcf --lang zh-CN
npx zcf init --lang zh-CN

# 语言参数优先级：--all-lang > --lang > 用户偏好 > 交互提示
```

### 非交互模式（CI/CD 友好）

```bash
# 缩写版
npx zcf i -s -g zh-CN -t api_key -k "sk-xxx" -u "https://xxx.xxx"

# 完整版
npx zcf i --skip-prompt --all-lang zh-CN --api-type api_key --api-key "sk-xxx" --api-url "https://xxx.xxx"
```

## 开发最佳实践

### 工作流选择指南

- **小任务修复**：直接使用 `<任务描述>`，遵循 SOLID、KISS、DRY、YAGNI 原则
- **新功能开发**：使用 `/feat <任务描述>`，结构化开发流程
- **复杂项目**：使用 `/workflow <任务描述>`，完整六阶段开发流程
- **企业级项目**：使用 `/bmad-init` 初始化 BMad 工作流

### 配置建议

1. **项目初始化**：首次使用建议运行 `/init-project` 生成项目级 CLAUDE.md
2. **文档管理**：生成的文档保存在 `.claude/` 目录，建议加入 `.gitignore`
3. **风格选择**：根据团队偏好选择合适的 AI 输出风格
4. **备份策略**：定期备份重要配置，ZCF 会自动创建时间戳备份
